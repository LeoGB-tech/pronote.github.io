Questions fréquentes

Mes données sont-elles envoyées quelque part ?
Non. Pronote Plus fonctionne 100% localement dans ton navigateur. Rien n'est envoyé à un serveur. Tes notes restent sur ton appareil.
Pourquoi ça ne détecte pas mes moyennes ?
Tu dois absolument être sur l'onglet Notes → Par matière (pas "Par ordre chronologique"). Attends que la page soit complètement chargée, puis clique le favori.
Comment ça calcule les priorités ?
Pour chaque matière, le script simule l'effet d'un 20/20 sur ta moyenne générale. Là où tu as 2/20, un 20 crée un bond énorme. Là où tu as déjà 18/20, l'effet est minime. L'algo te donne les matières dans l'ordre d'impact décroissant.
Ça marche avec les coefficients ?
La détection des coefficients est en cours de développement. Actuellement, toutes les matières ont le même poids dans le calcul.
Comment fermer le widget ?
Clique sur la barre du haut (PRONOTE +) ou sur le ×. Pour le rouvrir, reclique sur le favori.
